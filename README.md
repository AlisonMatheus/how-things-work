# How Things Work

> "The important thing is not to stop questioning."
>
> — Albert Einstein

## About this project

**How Things Work** is not just a repository.

It is a long-term journey to understand how technology really works.

Instead of simply learning frameworks or following tutorials, every project in this repository starts with a single question.

- How does DNS find a website?
- How does Netflix stream videos?
- How does cloud gaming work?
- How does a computer communicate with another?
- How does NFC work?
- How does the cloud actually work?

Each folder is an investigation.

Every investigation contains experiments, code, documentation and conclusions built from scratch.

The goal is not only to write code.

The goal is to understand the systems behind the code.

---

## Philosophy

This repository follows a few simple principles.

- Learn by building.
- Learn by experimenting.
- Never copy code without understanding it.
- Always start with a question.
- Document everything.
- Stay curious.
